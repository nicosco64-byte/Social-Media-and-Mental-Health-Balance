# Social-Media-and-Mental-Health-Balance
Excel dashboard project analyzing the relationship between social media usage and mental health indicators such as stress, sleep quality, and happiness. Visualizes trends and correlations using a cleaned Kaggle dataset; includes interactive charts and slicers for demographic filters.

This project features an interactive Excel dashboard analyzing the relationship between social media usage and key mental health indicators (stress, sleep quality, and happiness). The analysis is based on a cleaned dataset from Kaggle and visually explores how different factors are correlated.


Visualizations of how social media usage, screen time, and days spent off social media relate to stress, sleep, and happiness.

Interactive slicers to filter results by demographic variables (gender, age group).

Summary statistics, trendlines, and correlation analysis.

Clear layout with consistent formatting for an attractive, easy-to-read dashboard.


Download the Excel file (.xlsx format) from this repository.

Open the file in Microsoft Excel (recommended) to view all charts, pivot tables, and slicers.

Use slicers to filter by demographic groups and explore different data segments.

Review the insights summary below each visualization for key findings.
